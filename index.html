Guiding the Creation and Generating a Custom Portfolio Website Using LLM AI

AI Tool Ideal Use Case

Step 1

Want to create your own portfolio? Don't know where to start?

Simply enter what type of designer you are to generate a portfolio ready for customisation.

A template will be created to follow along and help guide content creation for your portfolio.

Step 2

Complete the template to fill out the content side of the portfolio.

This can range from short taglines to what information to include for your projects.

Examples are provided to help ideation. Remember to add your personal touch to the content.

Step 3

Your AI generated portfolio website is ready!

Now it's time to add the content from the template and start adding your personal touch to the base website!

Process

LLMs Used

Gemini Advanced and v0 by Vercel were used in this AI tool process.

Gemini Advanced is used for language responses and checking the code generated by v0. v0 is an LLM that generates website components based on React and Tailwind CSS. The prompts used with v0 were generated by Gemini.

Through an iterative process, AI responses used in the tool were graded and improved upon.

On the left is the first baseline grading of the AI process.

Iteration Improvements

On the left is the final iteration grading.

Gemini Advanced quickly improved in the first step of the tool with more specific prompting such as providing previous successful prompts. In the later half, I could not get it to check and change code consistently after the components were generated with v0.

This AI tool process works it's just not consistent with its results.

The most successful attempt was used for step 3 to give an ideal output from the process.

Final Thoughts

Gemini Advanced not being able to process code consistently was a surprise, as it has been shown to excel at coding tasks.

Designing a portfolio is quite a subjective process, v0 seems to get around this issue by providing 3 generated options to choose from and allowing iterations on the same prompt (See Image). Perhaps this process could benefit something similar to give a more tailored portfolio.

The code generated by v0 is using Next.js and React which isn't as simple to use as traditional HTML and CSS. It may have been better to try and generate this traditional code, however, v0 can create complex components with very few prompt instructions.
